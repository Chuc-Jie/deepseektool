## DeepSeek 功能增强工具箱 v4.1.1

### 一、脚本概述

本脚本为 DeepSeek Web 端提供一系列增强功能，所有管理项集成在**统一控制面板**中，设置后即时生效，无需刷新页面。

包含三大核心模块：

1. **代码块折叠**：按行数阈值自动折叠，可自定义预览行数，手动展开不受后续加载重置。
2. **表格优化与导出**：修复表格宽度问题，可选的 PNG / CSV 导出按钮。
3. **AI 思考过程自动折叠**：自动折叠"已思考"区域，支持模拟点击和 CSS 直接折叠两种模式。

### 二、功能详情

#### 2.1 代码块折叠
- 在代码块右上角添加"折叠/展开"按钮（含箭头图标）。
- **自动折叠阈值**：代码行数超过设定值时自动折叠（默认 20 行，设为 0 则禁用）。
- **折叠预览行数**：折叠后保留前 N 行可见（默认 0 表示完全隐藏）。
- **状态保持**：用户手动展开的代码块不会因后续对话加载而自动折叠。

#### 2.2 表格优化与导出
- **样式修复（始终生效）**：表格宽度自适应、列宽均分、圆角边框、行悬停高亮，深色主题跟随。
- **导出按钮（可开关）**：表格右下角悬浮 📸（PNG）和 📄（CSV）按钮，鼠标悬停显示。PNG 依赖 `html2canvas`，CSV 原生 UTF‑8 BOM。

#### 2.3 AI 思考过程自动折叠
- 自动寻找"已思考"区域并折叠，避免页面过长。
- 两种折叠模式：
  - **模拟点击**：通过点击原生箭头折叠，保持与 DeepSeek 原生交互一致。
  - **CSS 直接折叠**：关闭模拟点击后使用，直接隐藏思考内容区域。

### 三、使用说明

1. 安装脚本并访问任一 DeepSeek 对话页面。
2. 点击 **Tampermonkey / ScriptCat 菜单 → "⚙️ 脚本设置"** 打开控制面板。
3. 可调整：代码块折叠阈值、预览行数、表格导出按钮开关、AI 思考自动折叠及模式。
4. 修改设置后**立即生效**，无需刷新。
5. 配置自动保存，下次打开页面保持。

### 四、兼容性

- 支持域名：`chat.deepseek.com`、`www.deepseek.com`、`deepseek.com`
- 依赖：`html2canvas@1.4.1`（`@require` 引入）
- 基于 DeepSeek 当前 DOM 结构编写，若页面大版本更新可能导致部分功能失效。
